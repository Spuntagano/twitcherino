## Getting Started


Create the configuration file "server/config/coffee/config.coffee" and override what you need.


### Dependencies

Install the node dependencies

```shell
npm install
```

Install the front-end dependencies

```shell
bower install
```

Compile Sass, coffeescript and start the watch

```shell
grunt
```
